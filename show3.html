Media Player — 設定說明
⚠️
您正在從 file:// 直接開啟此檔案。
瀏覽器安全限制導致 AVI / MKV / MOV 轉碼功能（FFmpeg）無法啟用。
MP4、WebM、圖片等格式仍可正常使用。
若需要轉碼功能，請使用下方任一方式以 本地伺服器 開啟。

選擇啟動方式

把 start.bat（同資料夾）雙擊執行，或手動執行：
start.bat點擊複製
執行後瀏覽器會自動開啟 http://localhost:8765
把 start.sh（同資料夾）在終端執行：
bash start.sh點擊複製
執行後瀏覽器會自動開啟 http://localhost:8765
在此 HTML 所在資料夾開啟終端，執行：
python -m http.server 8765點擊複製
然後開啟瀏覽器前往 http://localhost:8765
⚠ 注意：Python 內建 server 不支援 COOP/COEP header，FFmpeg 仍可能無法啟用。建議用 start.bat/sh。
安裝並執行 serve：
npx serve . -p 8765 --cors點擊複製
然後開啟 http://localhost:8765
不需要 AVI/MKV 轉碼，只播 MP4 / 圖片
